# IPC Subclass 完整生命週期地圖

> **用途**：追蹤每一個 IPC Subclass 在哪個版本出現、在哪個版本消失、映射到哪裡
> **資料覆蓋**：IPC v6（1994）→ IPC 2026.01，全 24 份官方 PDF 已讀完
> **程式對應**：`pipeline/ipc_concordance.py`
> **資料庫範圍**：民國 85–114 年（1996–2025）
> **最後更新**：2026-04-03

---

## 凡例

| 符號 | 意義 |
|------|------|
| `≤ 1994` | 第 6 版（1994）即已存在 |
| `YYYY.NN` | 該版本首次出現 / 正式廢止 |
| ✅ **現行** | 截至 2026.01 仍有效 |
| ❌ **廢止** | 已正式廢止，DEPRECATED_TO 有記錄 |
| 🔀 **重組** | Subclass 未廢止但有大量 group 遷入或遷出 |
| `[PDF✓]` | 有官方 PDF 直接確認 |
| `[inferred]` | 依技術內容推斷（2016–2022 的全新空白 subclass） |
